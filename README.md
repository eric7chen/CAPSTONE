# P1 - ParseCsv
Simple project to parse the csv's from the airtable language bases with unit tests

## Requirements
openjdk-11
Apache Maven

## Build and Deploy
To build the target directory, run
```{bash}
> mvn package
```
To clean the target directory, run
```{bash}
> mvn clean
```
To run all tests, run
```{bash}
> mvn test
```
